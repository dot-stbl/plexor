P Plexor cloud console · prototype

Облачная консоль. Шесть экранов, которые показывают, что внутри.

Перед тобой прототип UI для self-hosted облачной платформы Plexor. Каждый экран — самостоятельная страница продукта: реальные таблицы, работающие табы и wizard, плотный grid, тонкие бордеры, моноширинные числа. Открой любую карточку, чтобы посмотреть.
