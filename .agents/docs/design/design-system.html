Дизайн-система Plexor

Живой справочник консоли self-hosted облака. Одно направление — tech / utility: плотная сетка, тонкие бордеры, моноширинные числа, монохромная палитра и только статусные цвета. Всё на этой странице собрано из того же styles.css, что и экраны прототипа — правки в токенах отражаются здесь автоматически.

Плотность — это фича
Больше информации на экран, меньше украшений. Dmitriy читает таблицу, а не любуется ей.
Монохром + статусы
Серая шкала как база. Зелёный / красный / жёлтый — только для смысла: running, error, paused.
Числа — моноширинные
IP, ID, метрики, деньги — JetBrains Mono с табличными цифрами. Колонки выравниваются сами.
